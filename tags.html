Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via tables and related tags

The tables below don't include all the HTML tags, but they do list many of the most important ones.

Documents Tags

Tag Name Purpose
HTML HTML root element that designates the document as HTML
head head represents a collection of metadata for the document
title title defines the document's title or name that is shown in a browser's title bar or a page's tab
meta meta represents metadata that cannot be represented by other HTML meta-related elements
body body represents the content of the document
header header represents the introductory section of a page or section

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
ol ordered list make a sequential list
ul unordered list make a bulleted list
li line item make a item in a list
div division creates an otherwise invisible block level container meant to be styled
blockquote block quotation creates an indented container for quotations

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="https://learnenough.com">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code make monospaced text <code><meta name="viewport" content="width=device-width"></code> <meta name="viewport" content="width=device-width">
span span creates an invisible inline container otherwise meant to be styled <span>This text is contained in a span<span> This text is contained in a span
